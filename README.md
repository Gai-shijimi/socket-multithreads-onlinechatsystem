# オンラインチャットシステム（マルチスレッド）
TCP通信とUDP通信を使用したオンラインチャットシステム

# 概要
  TCP通信でルーム名・ユーザー名・部屋を作る/参加するモード等を選択すると（今回はプログラム上にすでに記載した）、サーバー側でクライアントのトークンを管理するようになり、トークンをクライアントに渡す。そして受け取ったトークンを使用してクライアントは指定した部屋で他のユーザーと通信できる仕組み。

## 苦労した点
サーバー側において、スレッドを使用して受信と送信を並列に処理させる構造の作成とクライアントをどう管理するかが主に苦労した。

## 改善したいこと
クライアントがチャットを正常に終了できるようサーバー側の処理をどのようにしていくかということ。
